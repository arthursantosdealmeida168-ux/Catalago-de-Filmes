**Bem-vindo ao seu projeto Base44**

**Sobre**

Visualize e edite seu aplicativo em [Base44.com](http://Base44.com)

Este projeto contém tudo o que você precisa para executar seu aplicativo localmente.

**Edite o código em seu ambiente de desenvolvimento local**

Qualquer alteração enviada para o repositório também será refletida no Base44 Builder.

**Pré-requisitos:**

1. Clone o repositório usando a URL Git do projeto
2. Navegue até o diretório do projeto
3. Instale as dependências: `npm install`
4. Crie um arquivo `.env.local` e defina as variáveis ​​de ambiente corretas

```
VITE_BASE44_APP_ID=seu_id_do_aplicativo
VITE_BASE44_APP_BASE_URL=sua_url_de_backend

Exemplo:

VITE_BASE44_APP_ID=cbef744a8545c389ef439ea6
VITE_BASE44_APP_BASE_URL=https://my-to-do-list-81bfaad7.base44.app
```

Execute o aplicativo: `npm run dev`
